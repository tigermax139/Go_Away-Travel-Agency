// Modal window
Для того что бы сделать модальное окно
Кнопке которая будет его вызывать задайте:
 1)id
 2)data-target который равный значению id вашего окна

<button class="callback__btn" id="modalCallback-b"
data-target="#modalCallback-c">Заказать звонок</button>

3) модальному окну задайте  id который должен быть равен атрибуту data-target у кнопки,
которая его открывает

<div class="modal container-fluid " id="modalCallback-w">

4)в модальном окне создайте кнопку с классом modal-close
она автоматически розместится в правом верхнем углу и по ёё нажатию окно буде закрыватся

<button class="callback__btn modal-close"> </button>

5)подключите файл main.js
6)скриптом ниже инифиализируйте модальное окно
вызвав функцию modalWindow(),
                       ВНИМАНИЕ!!!
первым параметром передайте id кнопки что открывает окно, а вторым параметром id окна,
которое она должна открывать

modalWindow('#modalCallback-b', '#modalCallback-w');

Окон может быть не ограниченое количество, только коректно указывайте id елементов
не забывайте, что id любого елемента ВСЕГДА уникален,
